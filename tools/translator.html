Mode hors ligne

Le traducteur nécessite une connexion Internet pour fonctionner. Certaines fonctionnalités peuvent être limitées en mode hors ligne.

Aide - Traducteur

Comment utiliser

  1. Sélectionnez la langue source ou utilisez la détection automatique
  2. Choisissez la langue cible dans laquelle vous souhaitez traduire
  3. Saisissez ou collez votre texte dans la zone de gauche
  4. La traduction s'affiche automatiquement dans la zone de droite
  5. Utilisez les boutons pour copier, écouter ou partager la traduction

Fonctionnalités

  • Traduction automatique : les traductions sont effectuées dès que vous arrêtez de taper
  • Dictionnaire : cliquez sur un mot pour voir sa définition et ses synonymes
  • Prononciation : écoutez la prononciation du texte source et de la traduction
  • Exemples d'utilisation : découvrez comment les mots sont utilisés en contexte
  • Historique : retrouvez facilement vos traductions précédentes

Limitations

Cet outil utilise des API de traduction externes lorsqu'il est en ligne. La précision des traductions peut varier selon les langues et la complexité du texte. Pour les documents officiels ou les traductions professionnelles, il est recommandé de consulter un traducteur humain.

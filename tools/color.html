Aide - Sélecteur de couleur

Comment utiliser

  1. Cliquez et déplacez le curseur sur le carré coloré pour sélectionner une couleur
  2. Utilisez le curseur horizontal pour changer la teinte de base
  3. Ajustez l'opacité avec le curseur d'alpha si nécessaire
  4. Les valeurs HEX, RGB, HSL et CMYK sont automatiquement calculées
  5. Utilisez les onglets pour explorer les palettes, harmonies et autres outils

Fonctionnalités

  • Palettes : générez automatiquement des teintes, nuances et tons de votre couleur
  • Harmonies : découvrez des combinaisons de couleurs harmonieuses (complémentaires, analogues, etc.)
  • Ajustement : modifiez précisément la teinte, saturation, luminosité et température
  • Accessibilité : vérifiez le contraste pour la lisibilité et visualisez l'apparence pour différents types de daltonisme
  • Historique : gardez une trace des couleurs précédemment sélectionnées

Astuces

  • Cliquez sur l'icône de copie à côté d'une valeur pour la copier dans le presse-papiers
  • Vous pouvez saisir directement une valeur HEX dans le champ correspondant
  • Pour une interface accessible, visez un ratio de contraste d'au moins 4.5:1 (AA)
  • Utilisez la simulation de daltonisme pour vérifier que vos couleurs sont perceptibles par tous

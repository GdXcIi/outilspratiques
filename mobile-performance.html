Optimisation des performances sur mobile

Notre site a été optimisé pour offrir une expérience fluide sur tous les appareils, y compris les mobiles qui ont souvent des ressources limitées.

20 FPS

Avant optimisation

Animations saccadées sur mobile avec des pics de consommation CPU, entraînant une baisse de la durée de vie de la batterie.

60 FPS

Après optimisation

Animations fluides, contrôlées et efficaces, avec une consommation d'énergie réduite pour préserver la batterie.

Améliorations apportées

  • Limitation intelligente du FPS : Les animations sont limitées à 60 FPS sur mobile pour assurer une fluidité optimale tout en préservant les ressources.
  • Détection automatique : Le système identifie les appareils mobiles et adapte automatiquement les performances.
  • Animations optimisées : Toutes les animations (métronome, minuteurs, notifications) utilisent le nouveau système d'optimisation.
  • Réduction de la consommation d'énergie : Moins de calculs inutiles = meilleure autonomie de la batterie.

Comment ça marche ?

Notre nouveau PerformanceManager régule intelligemment le taux de rafraîchissement des animations :

// Limitation intelligente du FPS sur mobile
frameRateLimiter(animationFunc, targetFPS = 60) {
  const isMobile = /Android|iPhone|iPad/i.test(navigator.userAgent);
  
  // Pour mobile, limiter à 60 FPS
  if (isMobile) {
    let lastTime = 0;
    const interval = 1000 / targetFPS;
    
    const animate = (timestamp) => {
      const deltaTime = timestamp - lastTime;
      if (deltaTime >= interval) {
        lastTime = timestamp;
        animationFunc(timestamp);
      }
    };
  }
}

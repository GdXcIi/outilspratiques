Aide - Chronomètre

Comment utiliser

  1. Cliquez sur le bouton "Démarrer" pour lancer le chronomètre
  2. Le même bouton se transforme en "Pause" pour suspendre temporairement le chronométrage
  3. Utilisez le bouton "Tour" pour enregistrer des temps intermédiaires
  4. Le bouton "Réinitialiser" remet à zéro le chronomètre
  5. Vous pouvez effacer les tours sans réinitialiser le chronomètre

Raccourcis clavier

  • Démarrer/Pause Espace
  • Enregistrer un tour L
  • Réinitialiser R
  • Effacer les tours C
  • Plein écran F
  • Afficher/Masquer l'aide H

Fonctionnalités

  • Tours : enregistrez des temps intermédiaires pour mesurer vos performances
  • Statistiques : visualisez votre meilleur tour, tour le plus lent et moyenne
  • Export : sauvegardez vos résultats en CSV pour analyse
  • Précision : mesure précise au centième de seconde
  • Mode plein écran : pour une meilleure visibilité à distance

Utilisations courantes

  • Entraînement sportif
  • Tests de performance
  • Minutage d'activités
  • Expériences scientifiques
  • Présentations chronométrées
